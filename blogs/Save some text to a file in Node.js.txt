NodeJs Tut 1
So you want to save some text to a file using Node.js.

Super simple:

import fs from 'node:fs'

const text = 'yo'

fs.writeFile('text.txt', text, (err) => {
  if (err) {
    console.error(err)
  }
  console.log('done')
})
